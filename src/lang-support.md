# 语言支持

支持以下语言和语言服务器。要使用语言服务器功能，你必须首先安装 [lsp][LSP-Install-wiki]。

使用 `hx --health` 检查你安装的 Helix 版本中的语言支持。

有关语言配置的更多信息，请参阅 [语言配置][lang-config] 文档和 [添加语言][adding-languages] 指南。
